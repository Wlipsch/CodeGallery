“Unlock the Power of Code: Embrace the Future of Innovation and Possibility”

Belang van code

Het leren van code, zelfs op jonge leeftijd, helpt kinderen bij het ontwikkelen van probleemoplossende vaardigheden, logisch denken en creativiteit.

In de moderne wereld neemt code een steeds belangrijkere rol in. Het is de sleutel tot het begrijpen en beheersen van technologie. Met code kunnen we complexe systemen ontwikkelen, software creëren en digitale ervaringen tot leven brengen. Het belang van code strekt zich uit tot verschillende gebieden, van communicatie en entertainment tot wetenschap en bedrijfsvoering.

Kortom, het belang van code kan niet worden onderschat. Het helpt ons om de wereld om ons heen vorm te geven en de uitdagingen van de toekomst aan te gaan.

De invloed van code op het onderwijs

In het klaslokaal is steeds meer technologie te vinden. Is dit wel goed voor de bevordering van het onderwijs? In deze documentaire gaat Tuur op pad met de vraag: hoe kan de digitalisering het traditionele onderwijs veranderen? De digitalisering in het klaslokaal heeft natuurlijk alles te maken met code.

Workshop

Om kinderen kennis te laten maken met code bieden wij een workshop aan die de kinderen de kans geeft om op een leuke en interactieve manier de basisbegrippen van code te leren. De workshop bestaat uit 3 sessies. Tijdens de sessies ontwikkelen ze hun creativiteit en probleemoplossend vermogen. Aan het einde van de derde sessie hebben de kinderen een eigen kunstwerk gemaakt.

Sessie 1

In het eerste deel van deze sessie worden de kinderen geïntroduceerd met de belangen van code en hoe ze dit in het dagelijks leven kunnen tegenkomen. Na deze korte introductie met een quiz is het tijd om ideeën op papier te zetten waarna een begin wordt gelegd aan het coderen.

Sessie 2

In de tweede sessie maken de kinderen verder kennis met coderen. De kinderen starten onder begeleiding met het maken van verschillende vormen en kunstwerkjes. Aan het einde van de sessie wordt er een begin gemaakt aan hun eindwerk.

Sessie 3

In de laatste sessie gaan de kinderen aan de slag en realiseren ze een creatief kunstwerk met de kennis die ze uit de vorige delen hebben opgedaan.
